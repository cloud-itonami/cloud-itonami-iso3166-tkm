# Operator guide — TKM

Portal: no verified national e-procurement portal for Turkmenistan was
found this iteration -- this actor's market-entry mechanism is
business registration plus public-procurement compliance.

Entity registration: Ministry of Justice, Department of state
registration of legal entities (minjust.gov.tm) -- moderate confidence,
sourced from secondary/commercial material plus one state-affiliated
news outlet, not independently read from the primary government site
this session.

Procurement compliance: Law on Tenders for the Supply of Goods, Works
and Services for State Needs (adopted 20 December 2014, effective
1 July 2015) -- high confidence, this catalog's strongest fact.

Tax registration: State Tax Service, *hususy salgyt belgisi* (HSB),
12-digit Taxpayer Identification Number -- lower confidence, sourced
only from expat/freelancer tax-advisory blogs, not government-sourced.

All three checklist items must be on file (via `:jurisdiction/assess`)
before a `:filing/draft` proposal can pass the governor's
evidence-incomplete check. Human-gated filing only -- see README
Actuation.
